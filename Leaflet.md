A lightweight, open-source JS library for interactive web maps.
- *"Leaflet is jQuery for web maps — not the flashiest, but simple, reliable, 
  and everywhere."

Renders a [[Slippy Map]] in the browser (tiles, markers, popups, vector overlays).

  Small (~42KB), simple API, does the 80% case extremely well. Deliberately minimal — no built-in data analysis, no 3D, no advanced projections. Everything beyond basics is a plugin.

Still the default choice for "I need a map on a webpage"; huge install base, tons of tutorials, works everywhere.
- [[Mapbox GL JS]] has taken over for anything needing [[Vector Tile]]s or 3D.
- For heavy data visualization, [[deck.gl]] is better.
