A **dynamically typed** language checks types at runtime. Variables are not bound to a fixed type; the same name can hold values of different types over the course of execution.

Examples: Python, JavaScript, Ruby, Lisp, PHP.

Trade-offs:
- Faster to prototype, more flexible, less boilerplate.
- Type errors only surface when the offending code actually runs, so bugs may slip into production unless covered by tests.
