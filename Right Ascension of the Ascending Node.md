An [[Orbital Elements|Orbital Element]], part of a set of parameters describing the orbit of an object around a central body.


Tells you where the orbital plane is rotated around earth.
- First, define the ==line of nodes,== where the satellite's orbital plane crosses (e.g.) the Earth's equatorial plane.
- This line crosses the actual object's orbit at two points:
	- ==[[Ascending Node]]==: The point on the orbit where the satellite crosses the reference plane going north.
	- ==[[Descending Node]]==: The point on the orbit where the satellite crosses the reference plane going south.
- RAAN then is the *angle*, measured in the equatorial plane, from a fixed reference direction in space to the ascending node. It tells you which way the tilted orbital plane is pointed around Earth.
