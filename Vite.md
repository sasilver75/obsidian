Pronounced ("Veet", French word for "quick")
Javascript build tool/development server. Can turn modern syntax code like JSX, TypeScript, SASS into regular HTML, CSS, and JavaScript that browsers understand.
It's a replacement for `create-react-app`.

Two major parts:
- Dev server that serves source files over native ES modules, with rich built-in features and astonishingly fast [[Hot Module Replacement]]
- A build command that bundles your code with [[Rollup]], pre-configured to output highly-optimized static assets for production.

Also extensible via its plugin API and JS API with full typing support.
