A [[Vegetation Index]] ([[Spectral Index]]) similar to NDVI, but using the red edge band instead of standard red.

```
NDRE = (NIR - RedEdge) / (NIR + RedEdge)
```


Vs. [[Normalized Difference Vegetation Index|NDVI]]:
- NDVI uses red (~660nm) and [[Near Infrared|NIR]] (~840nm)
- NDRE uses red edge (~720nm) and NIR
