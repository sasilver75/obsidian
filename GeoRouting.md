
The broad practice of directing user traffic to a specific region, data center, edge location, or service endpoint based on geographic location and routing policy. 

It can be implemented using [[Domain Name Service|DNS]], [[Content Delivery Network|CDN]] edge logic, global load balancers, [[Anycast]]/[[Border Gateway Protocol|BGP]], or application-level redirects.


GeoRouting, is a broader concept than [[GeoDNS]], which is a specific implementation changes DNS answers based on location (specifically, the [[Authoritative Nameserver]] returns the closest (e.g.) [[A Record]] to the [[Recursive Resolver]]'s IP); GeoRouting is the *overall traffic-steering behavior,* regardless of where it is implemented.
