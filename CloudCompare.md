Open source desktop application for point cloud and mesh visualization, analysis, and processing. The go-to tool for:
- Visualizing and inspecting point clouds interactively
- Manual classification editing
- Distance computation between two point clouds (M3C2 algorithm, used for change detection, erosion monitoring)
- ICP (Iterative Closest Point) registration, for aligning two point clouds
- Roughness, curvature, and noise estimation
- Quick format conversion

Not a pipeline tool, more like an interactive analysis and quality checking environment. Essential for ground-truthing automated results.
