Formerly known as PrestoSQL.

A fast, distributed SQL query engine designed for interactive analytics on large datasets, letting users query data where it lives without needing data movement or centralization.

It supports querying across heterogenous data sources, including:
- [[Data Lake]]s ([[Amazon S3|S3]])
- SQL databases
- NoSQL databases

... all using standard SQL.
