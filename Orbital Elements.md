A set of parameters used to describe the size, shape, orientation, and current position of an object's (e.g. [[Satellite]]'s) orbit around a central body.

They are:
- [[Semi-Major Axis]] (a): Defines the size of the orbit
- [[Eccentricity]] (e): Defines the shape of the orbit, from circular to elliptical
- [[Inclination]] (i): Defines the tilt of the orbit relative to the reference plane, usually Earth's equator
- [[Right Ascension of the Ascending Node]] (RAAN; Ω): Define where the orbit crosses the equatorial plane going north
- [[Argument of Periapsis]] (ω): Defines where the closest point of orbit is, within the orbital plane
- [[True Anomaly]]: Define where the object is along the orbit, at a specific time

Sometimes [[Mean Anomaly]] (M) is used instead of True Anomaly, especially is propagated orbit data, because it is mathematically convenient.

To fully define the object's positions, the elements need a epoch, meaning the exact time at which those elements are valid.
