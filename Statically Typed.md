A **statically typed** language checks types at compile time, before the program runs. Variables are bound to a type, and type errors surface as compilation failures rather than runtime crashes.

Examples: C, C++, Java, Rust, Go, Haskell, TypeScript.

Trade-offs:
- Catches type errors early, improves tooling (autocomplete, refactoring), and enables compiler optimizations.
- More upfront ceremony; types must be declared or inferred.
