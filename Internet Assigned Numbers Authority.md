The organization that coordinates key global Internet identifiers (naming, numbering systems) so that different networks and software can interact and the Internet works consistently worldwide.

Responsibilities:
- IP address allocation: Coordinates blocks of IPv4 and IPv6 addresses, which are then distributed through [[Regional Internet Registry|Regional Internet Registries]].
- DNS root zone management: Helps manage the authoritative list of [[Top-Level Domain]]s (TLDs) like `.com`,`.org`, etc.
- Protocol parameter registries: Maintains official numbers used by Internet protocols, such as [[Port]] numbers, [[Multipurpose Internet Mail Extensions|MIME Type]], and protocol IDs.




The rough flow is:

[[Internet Assigned Numbers Authority|IANA]] → [[Regional Internet Registry|RIR]]s → [[Internet Service Provider|ISP]]s/cloud providers/large networks → end users and organizations

For example, IANA allocates a large block of IP addresses to ARIN, ARIN allocates smaller blocks to Comcast or [[Amazon Web Services|AWS]]
