A Python library letting do some different components with [[Raster]] analytics in a Notebook/python setting.

Written in Rust with Python bindings. Strong for:
- Hydrological analysis from [[Light Detection and Ranging|LiDAR]]-derived [[Digital Elevation Model|DEM]]s
- Terrain attributes (slope, aspect, curvature, hillshade)
- LiDAR ground filtering
- DEM void filling
- Geomorphemetric analysis
