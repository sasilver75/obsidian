An open-source electronics platform for building small embedded systems.
Usually means two things:
- Arduino boards: Single microcontroller boards like the Arduino Uno/Nano/Mega/etc.
- Arduino software ecosystem: The IDE, libraries, and simplified C/C++ programming model

You use Arduino to read sensors, control motors, blink LEDs, drive displays, log data, trigger relays, talk over serial/I2C/SPI and prototype embedded electronics.
