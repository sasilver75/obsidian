A type of [[Browser Storage]].

The Cache API stores [[HTTP]] `Request` / `Response` pairs.
- Commonly used with [[Service Worker]]s to make applications load faster or work offline.
- Useful for caching network resources rather than general-purpose application data.
- Use [[IndexedDB]] for structured application data.

GOOD for:
- Caching static assets
- Offline pages
- API response caching
- [[Progressive Web Application]]s
