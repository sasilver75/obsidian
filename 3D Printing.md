Building physical objects layer by layer from a digital model, a form of [[Additive Process|Additive Manufacturing]].

Workflow:
1. Create or download a 3D model, usually an `.stl`, `.obj`, or `.3mf` file
2. Slice the model with software that converts it into printer instructions
3. Print the object layer by layer
4. Clean up the result, such as removing supports, sanding, curing, or painting

Common 3D printing types:
- FDM/FDF: The most common hobbyist type. Melts plastic filament like PLA, PETG, ABS, or TPU. Good for prototypes, brackets, toys, enclosures, and functional parts.
- Resin/SLA/MSLA: Uses light to cure liquid resin. Very high detail, great for miniatures, jewelry models, dental parts, and detailed prototypes. 
- SLS/MJF: Industrial methods that fuse powder, often nylon. Stronger parts, typically no support structures, but more expensive.
- Metal printing: Used in aerospace, medical implants, tooling, and advanced manufacturing.

Common materials:
- PLA: Easiest, good for beginners, decorative and light functional parts.
- PETG: Tougher and more heat-resistant than PLA.
- ABS/ASA: Stronger and more heat-resistant, but needs ventilation and enclosure.
- TPU: Flexible rubber-like parts.
- Resin: High detail, but brittle unless using tough/speciality resins.

Terms:
- FDM: Fused Deposition Modeling; printing by melting and depositing plastic filament.
- FFF: Fused Filament Fabrication; Essentially the same process as FDM.
- PLA: Polylactic Acid; An easy-to-print plastic filament.
- PETG: Polyethylene Terephathalate Glycol: A tougher plastic filament.
- ABS: Acrylonitrile Butadiene Styrene; A strong, heat-resistant plastic.
- ASA: Acrylonitrile Styrene Acrylate; similar to ABS, with better UV/weather resistance.
- TPU: Thermoplastic Polyurethane; Flexible, rubber-like filament.
- SLA: Stereolithography Apparatus; Resin printing using light to cure liquid resin.
- SLA: Stereolithography Apparatus; resin printing using light to cure liquid resin.
- MSLA: Masked Stereolithography Apparatus; resin printing using an LCD mask and UV light.
- SLS: Selective Laser Sintering; fuses powder with a laser.
- MJF: Multi Jet Fusion; HP’s powder-based industrial 3D printing process.
