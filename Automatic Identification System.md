An automated, autonomous tracking system used on ships to prevent collision and aid navigation, operating via VHF radio. 

Broadcasts:
- Vessel's ID (==MMSI== number (Maritime Mobile Service Identity), Ship Name, ==IMO== number (International Maritime Organization))
- Position
- Course
- Speed
- Cargo type
To both ships and shore stations.

_________
### Aside: ==MMSI== vs ==IMO== Numbers
- Both vessel identifier numbrs
- IMO is a 7-digit number that's a *permanent, lifetime ID for a vessel for tracking and safety*. Stays with the ship for life regardless of name/owner/flag changes.
- MMSI is a 9-digit number that's a temporary ID for digital communication like AIS and Radio, and changes with flag and equipment.

_______

Mandatory (via the [[International Maritime Organization]]) for most ships over 300GT (Gross Tonnage; a measure of vessel's total internal volume, not its weight), it ensures safety in congested waters through real-time traffic monitoring.

While VHF is limited to line-of-sight (approximately 80 km), Satellite AIS (S-AIS) allows tracking of ships in open oceans.

AIS Classes
- Class A is used for large commercial vessels
- Class B is used for smaller, recreational boats

Vulnerabilities:
- Visibility: Not all vessels carry AIS, and it can be switched off
- Vulnerabilities: Susceptible to spoofing, jamming, and signal interference in high-traffic areas.
