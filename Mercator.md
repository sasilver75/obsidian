The Web Mercator is the de-facto standard for Web mapping applications. It rose to prominence when Google Maps adopted it in 2005.


![[Pasted image 20260411185815.png|400]]
Above: The Web Mercator projection is almost indistinguishable at global scale from Mercator projection cropped to around 85N to 85S. 
- The main difference is that the Web Mercator uses the spherical formulas at all scales, whereas large-scale Mercator maps generally use an ellipsoidal form of the projection. The discrepancy is imperceptible at global scale, but causes maps of local areas to deviate slightly from true ellipsoidal Mercator maps at the same scale.
