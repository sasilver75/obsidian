A [[Hash|Cryptographic Hash Function]], standing for "Secure Hash Algorithm 256-bit", part of the SHA-2 family of hash functions.

> SHA-256 takes an input of arbitrary size and deterministically produces a fixed-size 256-bit output called a hash, digest, or message digest.

The output is 256 bits, typically written as 64 [[Hexadecimal]] characters, because each hexadecimal character represents 4 bits: `9f86d081884c7d659a2feaa0c55ad015a3bf4f1b2b0b822cd15d6c15b0f00a08`
