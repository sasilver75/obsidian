Python GIS library

> Fiona stream simple feature data to and from GIS formats like [[GeoPackage]] and [[Shapefile]]. 
> Can read and write real-world data using multi-layered GIS formats...
> Depends on [[Geospatial Data Abstraction Library|GDAL]] but is different from GDAL's own (ugly) bindings to Python.

See also: [[Pyogrio]]
