An open source and free alternative to [[ArcGIS]] used by developer and researchers to:
- Load and inspect geospatial datasets (Shape, GeoJSON, PostGIS connections)
- Visualizing data on a map without writing any code
- Running spatial analysis interactively
- Debugging -- if you're not sure if your PostGIS data looks right, connect QGIS directly to your database to look at it!

Think of it as a  ==GUI for your spatial data==.
