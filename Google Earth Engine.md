Cloud computing platform for planetary-scale geospatial analysis, primarily used by researchers, scientists, and governments for large-scale environmental monitoring.

What it is:
- A massive catalog of satellite imagery and geospatial datasets ([[Landsat]], [[Sentinel]], [[MODIS]], etc.) going back decades
- A cloud computing environment where you can run analysis directly on that data without downloading it
- An API (JS and Python) for writing geospatial analysis that runs on Gooogle's infrastructure.
