An open-source, off-grid, decentralized mesh network built to run on affordable, low-power devices, without cell towers or internet.

Uses small electronic radios to send signals using a frequency that doesn't require a license, so anyone can do it without certification. Many of these small radios connect to each other, forming a "mesh" of interconnected devices.

You can really only send small things like texts and sensor data, not complicated things like voice and video.
