In the context of [[Tile]] serving and web maps, refers to the full set of XYZ tiles across zoom levels (what's stored in [[MBTile]], [[PMTiles]], etc.)

In a raster/[[Cloud-Optimized GeoTIFF|COG]] context, refers to the overview levels embedded in a [[GeoTIFF]], also called "image pyramids", or just "overviews."

Usually refers to the same underlying idea: Pre-computed lower-resolution versions so that you don't have to downsample on the fly at render time.
