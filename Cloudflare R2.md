Cloudflare's [[Blob Storage|Object Storage]] solution, letting developer store large amounts of unstructured data without the costly egress bandwidth fees associated with typical cloud storage services.

Can be used for:
- Storage for cloud-native applications
- Cloud storage for web content
- Storage for podcast episodes
- [[Data Lake]]s
- Cloud storage output for large batch processes, such as ML Model artifacts or datasets
