A Python package for interactive mapping and geospatial analysis with minimal coding in a Jupyter environment.
- Good for exploratory data analysis in notebooks.
- Helps you visualize a GeoDataFrame on a map with one line
- Can be used to inspect raster datasets, satellite imagery, COGs
- Teaching and demos

It is NOT:
- A production mapping library (it's for notebooks, EDA)
- A replacement for something like [[MapLibre GL JS]] + [[deck.gl]] in a frontend application.

"A geospatial swiss army knife"
- [[geopandas]]
- [[pydeck]]
- [[whitebox-tools]]

Various interactive tools that allow users to load vector and raster data onto the map without coding, and perform geospatial analysis directly within the leafmap user interface without writing a single line of code. Contains 500+ tools.
