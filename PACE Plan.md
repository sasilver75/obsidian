==Primary, Alternate, Contingency, and Emergency== (PACE) is a methodology used to build a communication plan.
- Identify the different stakeholders that need to communicate
- Determine, if possible, the bets four, different, redundant forms of communication between each parties.
- Ideally these four methods' potential failures will be isolated from eachother.

Primary: Typically the best, most reliable, most used, fastest, and highest bandwidth of the options.
Last: The most time-consuming, costly, and inconvenient.

The plan doesn't designate things such as the exact radio channel or talk group to be used if you're using a radio, rather the order in which you would plan to use the radio, then telephone, then email, then courier as means of communication.
