
Anycast is a ==one-to-one-of-many== routing method. Multiple servers across the globe share the exact same IP address. When a device requests data, the network's routing protocols automatically direct the traffic to the server that is topologically closest (or has the fastest path).

Idea: Even though many machines have the same (e.g.) [[Internet Protocol|IP]], the traffic only goes to the single nearest node.

Common Uses: [[Domain Name Service|DNS]] root servers and [[Content Delivery Network]]s to load balance traffic and reduce latency.
